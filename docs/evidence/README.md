# Evidencia

Acá van los tres documentos que justifican el producto, generados en la investigación previa. **Todavía no fueron copiados a esta carpeta**: están en las descargas de la conversación de investigación, y traerlos es un pendiente de la base documental (ver README de la raíz).

1. `informe-forge.md` — informe científico: práctica deliberada, patrones, transferencia, principios de aprendizaje.
2. `tier-list-entrenamientos-ajedrez.md` — evaluación de los ejercicios del mercado (Aimchess, Chessly, Chess.com, Lichess, Chesstempo, Chessable, CT-ART) con criterios explícitos.
3. `forge-diseno-app-ajedrez.md` — el documento de diseño que traduce la evidencia a decisiones de producto.

> **Nota:** estos archivos se generaron cuando el proyecto se llamaba FORGE y conservan ese nombre por trazabilidad. El nombre actual del proyecto es **ELOmax**; el contenido sigue vigente.

Cuando el PRD dice "ver evidencia", refiere a estos tres.
