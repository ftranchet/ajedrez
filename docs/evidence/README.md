# Evidencia

Acá viven los tres documentos que justifican el producto, generados en la investigación previa:

1. `informe-forge.md` — informe científico: práctica deliberada, patrones, transferencia, principios de aprendizaje, análisis crítico de las apps existentes y diseño del programa de entrenamiento.
2. `tier-list-entrenamientos-ajedrez.md` — evaluación de los ejercicios del mercado (Aimchess, Chessly, Chess.com, Lichess, Chesstempo, Chessable, CT-ART) con criterios explícitos, y los cinco huecos que ninguna app cubre.
3. `forge-diseno-app-ajedrez.md` — el documento de diseño que traduce la evidencia a decisiones de producto (el antecedente directo del PRD).

> **Nota:** estos documentos se generaron cuando el proyecto se llamaba FORGE y conservan ese nombre (en archivos y contenido) por trazabilidad. El nombre actual del proyecto es **ELOmax**; el contenido sigue vigente.

Cuando el PRD dice "ver evidencia", refiere a estos tres. Son documentos históricos: no se editan; si una decisión de producto se aparta de ellos, eso se documenta en el PRD o en un ADR, no acá.
